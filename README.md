# Nutrition Center Expense Tracker

A beautiful, web-based expense tracking application that runs entirely in your browser. No installation required, no server needed, and completely free to use!

## Features

- ✅ **Add Expenses** - Track date, category, description, amount, and notes
- ✅ **Edit & Delete** - Update or remove expenses easily
- ✅ **Search & Filter** - Find expenses by description, category, or notes
- ✅ **Statistics** - View total expenses, monthly expenses, and record count
- ✅ **Export Data** - Download your expenses as a CSV file
- ✅ **Local Storage** - All data is stored in your browser (no cloud needed)
- ✅ **Beautiful UI** - Modern, responsive design that works on any device

## How to Use

### Getting Started

1. **Open the app**: Simply double-click on `index.html` file, or right-click and select "Open with" your web browser (Chrome, Firefox, Edge, Safari, etc.)

2. **Add your first expense**:
   - Fill in the date, category, description, and amount
   - Optionally add notes
   - Click "Add Expense"

3. **Manage expenses**:
   - Click "Edit" to modify an expense
   - Click "Delete" to remove an expense
   - Use the search box to find specific expenses
   - Use the category filter to view expenses by category

4. **Export your data**:
   - Click "Export Data" to download a CSV file with all your expenses

## Troubleshooting

### Data not saving?
- Make sure your browser allows local storage (most do by default)
- Check if you're in private/incognito mode (data may not persist)

### Can't see the app properly?
- Make sure all three files (`index.html`, `styles.css`, `app.js`) are in the same folder
- Try refreshing the page (F5 or Ctrl+R)

### Want to reset all data?
- Open browser developer tools (F12)
- Go to Application/Storage tab
- Find "Local Storage" → your site
- Delete the `nutritionExpenses` entry
